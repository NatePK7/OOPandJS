Java Script Key word "this"

Rule 1 Global: Keyword this in the wild (outside of the declared object. Object has not been defined which contains the keyword this) Value refers to the global object which is the window object (browser)

Rule 2 object/ implicit: Keyword this inside of a declared object value is the closest parent object

Rule 3 explicit using call, apply and bind only applies to functions: to explicitly set the value of Keyword this , we use call apply or bind. with the bind keyword, the first argument called is this

Rule 4 New: Keyword this with the new keyword, the this applies to the new empty object.
